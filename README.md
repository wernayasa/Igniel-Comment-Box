# Blogger Comment System (Clone Igniel Style)

Sebuah sistem komentar kustom untuk Blogger yang dirancang dengan inspirasi dari desain Igniel. Proyek ini memodifikasi struktur komentar bawaan Blogger menjadi tampilan yang lebih bersih, modern, dan interaktif.

## Fitur Utama
- **Desain Modern:** Menggunakan pendekatan minimalis yang fokus pada keterbacaan.
- **Responsive:** Tampilan tetap rapi baik di perangkat desktop maupun mobile.
- **Custom Badge:** Penanda khusus untuk Admin/Author (seperti bingkai "Wernayasa").
- **Optimasi Kecepatan:** Kode yang ringan untuk menjaga performa loading halaman.
- **User Friendly:** Penataan tata letak tombol reply dan hapus yang intuitif.

## Teknologi yang Digunakan
- **XML/HTML:** Untuk struktur dasar template Blogger.
- **CSS / Tailwind CSS:** Untuk penataan gaya dan efek glassmorphism.
- **JavaScript:** Untuk fungsionalitas interaksi dan manipulasi DOM.

## Cara Pemasangan

1. **Backup Template:** Selalu cadangkan template Blogger Anda sebelum melakukan perubahan.
2. **Cari Kode Komentar:** Temukan bagian `<b:includable id='threadedCommentJs' var='post'>` dan `<b:includable id='threadedComments' var='post'>` di editor HTML Blogger Anda.
3. **Ganti Kode:** Salin kode dari file `style.css` ke dalam template Anda.
4. **Simpan:** Klik simpan dan lihat hasilnya.

## Lisensi
Proyek ini dibuat untuk tujuan pembelajaran dan berbagi. Hak cipta desain asli tetap milik desainer aslinya (Igniel). Silakan gunakan dan modifikasi sesuai kebutuhan Anda.
